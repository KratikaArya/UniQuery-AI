### *A Conversational College Information Assistant*
About
---
**UniQuery AI** is a chatbot-based application that helps students access college information through natural language queries. Instead of searching across multiple websites, users can simply ask questions and receive instant answers about colleges.

The project focuses on institutions in Indore and supports multiple academic domains including Engineering, Commerce, Law, and Medical.

Purpose
---
College information is often scattered and difficult to compare. UniQuery AI organizes structured institutional data into a single interactive platform, making information access faster and easier for students and parents.

Features
---
* Answer college-related questions conversationally 
* Provide fee information 
* Show facility availability such as hostels, libraries, labs, etc.
* Display top-ranked colleges
* Enable domain-based exploration 
* Present results as chat responses or comparison tables
* Data caching to reduce repeated data loading and improve response performance
* Natural language query interpretation using rule-based NLP

How It Works
---
Users select a domain, ask questions naturally, and the system interprets the query to retrieve relevant information from the dataset and display results instantly.

Outcome
---
UniQuery AI provides a simple and interactive way to explore college information, reducing manual searching while improving accessibility and decision-making.
